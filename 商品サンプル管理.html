Firebase 初期設定

チームでデータを共有するための設定です

1

Firebase プロジェクトを作成

console.firebase.google.com を開いて「プロジェクトを追加」→ 名前を入力して作成

2

Firestore Database を有効化

左メニュー「Firestore Database」→「データベースの作成」→「テストモードで開始」→ロケーションは asia-northeast1（東京）を選択

3

設定コードを取得

⚙️ プロジェクトの設定 →「マイアプリ」→ </>（ウェブ）→ アプリ名を入力して登録 →「firebaseConfig」のコードをコピー

4

コードを下に貼り付けて保存

コピーした const firebaseConfig = { ... } の部分をそのまま貼り付けてください

設定は端末に保存されます。チームの全員がこの手順を行ってください。
GitHub Pages に公開すると URL をチームで共有できます。
